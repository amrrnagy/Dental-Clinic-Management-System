# Dental Clinic Management System 🦷

A professional desktop application built with Java and JavaFX to facilitate the daily operations of a dental clinic. This system provides a centralized platform for managing patient data, scheduling appointments, and tracking clinic finances.

## ✨ Features
- **Patient Management:** Create, update, and track detailed patient medical histories and records.
- **Doctor Scheduling:** Manage doctor availability and assign specialists to specific appointments.
- **Appointment System:** Intuitive interface for booking, rescheduling, and canceling dental visits.
- **Billing & Payments:** Automated ID generation for invoices and tracking of payment statuses.
- **Modern UI:** A clean, user-friendly interface designed with JavaFX and CSS for a smooth desktop experience.

## 🚀 Getting Started

### Prerequisites
- JDK 17 or higher
- Maven installed

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/amrrnagy/Dentist.git]
   ```
2.  Navigate to the project directory:
    ```bash
    cd Dentist
    ```
3.  Build the project:
    ```bash
    mvn clean install
    ```

### Running the Application

Execute the following command to launch the GUI:

```bash
mvn javafx:run
```

## 📂 Project Structure

  - `src/main/java`: Contains the core logic, divided into `model`, `view`, and `controller` packages.
  - `src/main/resources`: Contains `.fxml` layouts, images, and `.css` stylesheets.
